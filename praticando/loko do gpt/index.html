Linguagens importantes para o desenvolvimento web

HTML

O HTML é uma linguagem de marcação, que serve de base para a criação de sites. Com o HTML, é possível a criação, exibição, formatação e edição de uma página de internet, como:

  • Textos
  • Imagens
  • Formulários
  • Tabelas

JavaScript

A linguagem de programação JavaScript é usada para rodar scripts através dos navegadores. Com o JavaScript, algumas aplicações podem ser executadas sem a necessidade de troca de informações com o servidor. Assim, um documento pode ser alterado diretamente através dos scripts que interagem com o usuário. Iremos aprender JavaScript logo após estudar HTML.

PHP

A linguagem de programação PHP é do tipo interpretada e é, atualmente, uma das mais usadas no mundo. O PHP é vastamente utilizado para criar aplicações e sistemas do lado do servidor. Com PHP, é possível criar páginas com conteúdo dinâmico e interagir com servidores e bancos de dados.
