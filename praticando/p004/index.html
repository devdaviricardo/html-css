APRENDENDO PROGRAMAÇÃO
DE FORMA SIMPLES

Como primeiro exemplo irei usar a linguagem de marcação "HTML".

  1. HTML (HyperText Markup Language)
    2. é uma linguagem de marcação de hipertexto que define a estrutura e o conteúdo de páginas da web. É o principal bloco de construção da web.

Como segunda referência vou demonstrar a linguagem de estilo a "CSS"

  1. CSS (Cascading Style Sheets)
    2. é uma linguagem de estilização que define como os elementos de uma página da web devem ser exibidos. É uma ferramenta fundamental para o desenvolvimento web, permitindo criar sites atraentes e responsivos.

Por fim, para finalizar um bom site com interação, pode-se usar a linguagem de programção "JavaScript"

  1. JavaScript (JS)
    2. é uma linguagem de programação que permite criar páginas da web interativas. É uma das principais tecnologias da web, sendo usada para adicionar funcionalidades dinâmicas a sites e aplicativos.
